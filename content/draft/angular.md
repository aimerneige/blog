# Angular 入门

## 安装 angular

### 安装 nodejs

```bash
sudo apt install nodejs
```

### 安装 npm

```bash
sudo apt install npm
```

### 安装 angular/cli

```bash
npm install --global @angular/cli
```

## 构建一个 angular 项目

在任意目录下执行下面的指令即可新建一个 `myapp` 项目

```bash
ng new myapp
```

执行下面的指令启动这个项目

```bash
ng serve --open
```
